MASTER DATA API

Base URL:
https://wtpkpapi.citasys.net

Endpoint:
GET /api/masters/locations

Kegunaan:
Mengambil data master location untuk input lokasi atau location_code logsheet.

Butuh token:
Ya.

Headers:
Accept: application/json
Authorization: Bearer <token>

Query params:
Tidak perlu.

Body:
Tidak perlu.

Contoh request:
GET https://wtpkpapi.citasys.net/api/masters/locations
Accept: application/json
Authorization: Bearer <token>
